Control your Digitech effect pedal under Linux!
gdigi is tool aimed to provide X-Edit functionality to Linux users

Supported devices (list misses your DigiTech device? Please get in touch if you wish to add support for it!):

  • RP150
  • RP155
  • RP250
  • RP255
  • RP355
  • RP500
  • RP1000
  • GNX3000
  • GNX4K

Setup:
gdigi is available in Debian. To install issue following command:

sudo apt-get install gdigi

In order to compile gdigi on Debian/Ubuntu system, issue following commands:
sudo apt-get install build-essential libasound2-dev libgtk-3-dev libglib2.0-dev libexpat1-dev libxml2-dev
wget http://sourceforge.net/projects/gdigi/files/gdigi/0.4.0/gdigi-0.4.0.tar.bz2
tar xvjpf gdigi-0.4.0.tar.bz2
cd gdigi-0.4.0
make
./gdigi
